## Features

- **Responsive Design**: Optimized for all device sizes
- **Modern UI**: Built with React, TypeScript, and Tailwind CSS
- **Interactive Animations**: Smooth animations using Framer Motion
- **Service Showcase**: Detailed repair services and mobile sales
- **Product Gallery**: Visual showcase of shop interior, exterior, and team
- **Contact Integration**: Direct calling, emailing, and map directions

## Services Offered

- Screen Repair & Replacement
- Battery Replacement
- Software Updates & Optimization
- Charging Port Repair
- Water Damage Repair
- Full Device Diagnostics
- New & Second-hand Mobile Sales

## Technology Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with custom gradients
- **UI Components**: Radix UI + shadcn/ui
- **Animations**: Framer Motion
- **Build Tool**: Vite
- **Icons**: Lucide React

## Development

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone <repository-url>
cd mobile-hub-connect-main

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Scripts
- `npm run dev` - Start development server on http://localhost:8080
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
